# 玛纳之粮

到了晚上，有鹌鹑飞来，遮盖了营幕；到了早晨，营幕四周落了一层露水。露水升化之后，在旷野的地面上，留下稀薄的碎屑，稀薄得好像地上的霜。

——出16:13-14
